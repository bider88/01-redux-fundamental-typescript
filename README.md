# 01-redux-fundamental-typescript

Simple base of redux to understand the functionality and how it is made

### How to run
```
$ npm install
$ ts-node [file-name]
```
### Installation

Requires [Node.js](https://nodejs.org/) to run.
```
$ npm i -g ts-node
$ npm i -g typescript
```

### Dependecies
- Redux
```
$ npm i redux
```
